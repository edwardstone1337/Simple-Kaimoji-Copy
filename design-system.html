Page Background

CSS class: body

The page background is an animated conic gradient driven by --bg-gradient-angle and @keyframes rotate-bg-gradient, with dark mode switching to a static surface color.

Body background is visible around the page while this reference card remains readable.

Noise Overlay

CSS class: body::after

The subtle grain texture is rendered by a fixed body::after pseudo-element using an inline SVG turbulence pattern at low opacity.

Move around the page and inspect the viewport: the overlay stays fixed and does not intercept pointer events.
